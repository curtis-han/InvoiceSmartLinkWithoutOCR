# InvoiceSmartLink_withoutOCR

## 项目简介
InvoiceSmartLink_withoutOCR 是一个用于处理发票的工具，不依赖于光学字符识别（OCR）技术。该项目旨在提供高效、准确的发票数据处理解决方案。

## 功能特性
- **发票数据解析**: 从发票文件中提取关键信息。


## 安装指南
pip install requiements.txt
python main.py

## 使用方法
1. 修改JsonManger:
    - 修改年份
    - 根据自己的 勘定科目 进行修改
    ```
2. 将发票文件上传到指定目录，应用将自动处理并生成结果文件。
